# abner [![](https://travis-ci.org/creativelive/abner.svg?branch=master)](https://travis-ci.org/creativelive/abner)

Very basic hapi server for use in tests


## Usage

```
'use strict';

var server = require('abner')(/* opts */);
server.start();
```
